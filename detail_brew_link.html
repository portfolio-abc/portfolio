サイト名 BREW LINK（架空サービス）

種別
サブスクリプションサービスサイト（コーポレート要素含む）
ターゲット
20〜40代のコーヒー好きで、日常の中で「ちょっとした特別感」や体験価値を重視する層。
ペルソナ
都内在住・在宅ワーク中心の会社員。
毎朝コーヒーを飲む習慣があり、休日はカフェ巡りを楽しむ。
価格だけでなく「体験の質」や「ちょっとした特別感」に価値を感じる。
課題
コーヒーのサブスクリプションサービスは多く存在する中で、他サービスとの違いや体験価値が伝わりにくいという課題がある。
また、サービスの魅力が言語情報だけでは伝わりにくく、ユーザーが登録まで踏み切るための決め手に欠けている状態。
目的
サービスの魅力や体験価値を視覚的に伝え、ユーザーに「使ってみたい」と感じてもらうサイトを制作。
また、コーヒーを通じた豊かな時間や特別感を表現し、サービスへの興味・関心を高めることを目指した。
情報設計
ファーストビューでサービスの世界観と価値を直感的に伝え、その後「サービス理解 → メリット → 不安解消 → 行動」の流れを設計。
デザイン
コーヒーを楽しむ時間の「落ち着き」や「上質さ」を表現するため、余白を活かしたシンプルなデザインを採用。
また、写真や配色によって体験価値を視覚的に伝え、ユーザーが利用後のイメージを想起できるよう設計した。
さらに、情報を整理して視認性を高めるため、カード形式のデザインを多く取り入れ、直感的に内容を把握できる構成とした。
制作範囲
・企画 / 情報設計
・ワイヤーフレーム作成
・デザイン（Photoshop）
・コーディング（HTML / CSS）
・レスポンシブ対応
制作期間
・企画 / 設計：2日
・デザイン：4日
・コーディング：3日
使用ツール
・Visual Studio Code
・Photoshop
